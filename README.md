## Descrição
Um pequeno projeto de cadastro de usuários para treinar minhas habilidades com autenticação usando JWT e Passport.

## Env
Esse exercício está sendo executado numa máquina Lubuntu Linux, usando banco Postgres instalado via snap,
o app é feito em React, Tailwind, TypeScript e Nest com Node. O Nest está usando Express por baixo, como padrão
do framework.

## Setup do projeto

```bash
$ npm install
```

## Compilar e rodar o projeto

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Deployment

```bash
$ npm install -g @nestjs/mau
$ mau deploy
```
